## ğŸš€ Cara Penggunaan

Mengunduh konten sangatlah mudah:

1. Buka website Yoru-Downloader.
2. Pilih platform media sosial dari menu.
3. Tempelkan tautan (link) media ke kotak input yang tersedia.
4. Klik tombol **"Cari"** atau **"Konversi"**.
5. Pilih kualitas yang diinginkan (jika tersedia).
6. Klik tombol **Download** untuk mengunduh konten.
